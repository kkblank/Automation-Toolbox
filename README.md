# 自动化工具箱 (Automation-Toolbox)

一款基于 Python + PyQt6 开发的桌面端多功能工具集，提供 GUI 自动化、图像处理、记忆搜索、批量文件处理、OCR 文字识别等功能。

## 功能概览

| 功能模块 | 说明 |
|---|---|
| **GUI 自动化** | 通过 Excel 配置文件驱动鼠标键盘操作，支持图片识别点击、坐标点击、拖拽、长按、滚动、键盘输入等，可设置循环次数，支持 IF-ELSE-ENDIF 条件分支 |
| **图像处理** | 图像缩放重设，支持多种插值算法（快速/均衡/质量）和色彩模式（彩色/黑白），支持绝对像素和倍数缩放 |
| **记忆搜索** | 对本地文本文件库进行关键词搜索，支持多关键词匹配，可通过编号快速读取文件内容到剪贴板 |
| **批量处理** | 批量文件重命名（自定义文件头、位数、起始编号）+ 批量图片水印（文字/图片水印，支持位置和透明度调节） |
| **OCR 文字识别** | 基于 PaddleOCR 的图片文字识别，支持中文、英文、日文、德文、韩文、法文 |

## 环境要求

- Python 3.10+
- Windows 操作系统

## 安装与运行

### 方式一：源码运行

1. 安装依赖：

```bash
pip install -r requirements.txt
```

2. 运行主程序：

```bash
python main.py
```

### 方式二：使用打包版

直接运行 `dist/自动化工具箱/main.exe` 即可，无需安装 Python 环境。

## 使用说明

### 1. GUI 自动化

1. 在主界面点击「自动化」进入自动化模块
2. 点击「配置文件」打开 `data/配置文件.xls`，按格式编写指令
3. 如需使用图片识别功能，点击「图片文件夹」放置定位图片素材
4. 设置循环次数（0 表示无限循环），点击「开始」运行
5. 随时点击「结束」停止运行

**支持的指令类型：**

| 指令类型 | 说明 | 参数 |
|---|---|---|
| 图片 | 通过图片定位屏幕位置后点击 | 图片名、鼠标类型、点击次数、等待时间 |
| 坐标 | 直接点击指定坐标 | x/y 坐标、鼠标类型、点击次数、等待时间 |
| 拖拽 | 鼠标拖拽操作 | 起始坐标、结束坐标 |
| 长按 | 鼠标长按 | 鼠标类型、按住时长 |
| 滚动 | 滚轮滚动 | 滚动距离 |
| 键盘 | 键盘输入操作 | 操作类型、值、时长 |
| IF | 条件分支：检测图片是否出现 | 图片名、超时秒数 |

**辅助功能：**
- 点击「鼠标定位」可实时查看鼠标当前坐标
- 点击「单步调试」可执行指定行的一条指令进行调试
- 可调节图片检测速度（快/标准/慢）和图片匹配置信度

**IF-ELSE-ENDIF 条件分支用法：**

在 Excel 配置表中，新增以下指令类型实现条件分支：

| 说明 | 判断标志 | 图片/坐标 | 鼠标类型 | 点击次数 | 等待时间 |
|---|---|---|---|---|---|
| 检测弹窗 | **IF** | `popup.png` | | | **3** |
| IF 分支操作 | 图片 | `close.png` | 1 | 1 | |
| | **ELSE** | | | | |
| ELSE 分支操作 | 坐标 | `100/200` | 1 | 1 | |
| | **ENDIF** | | | | |

- **IF** 行：`图片/坐标` 列填要检测的图片名，`等待时间` 列填超时秒数
- 图片在超时内出现 → 执行 IF~ELSE 之间的指令块
- 超时未出现 → 跳转到 ELSE~ENDIF 之间的指令块（没有 ELSE 则直接跳到 ENDIF）
- ELSE 和 ENDIF 只需在 `判断标志` 列填入对应关键字
- 支持嵌套 IF

### 2. 图像处理

1. 在主界面点击「图像处理」进入图像处理模块
2. 选择插值算法（快速/均衡/质量）和色彩模式（彩色/黑白）
3. 输入目标尺寸（绝对像素如 `1920/720` 或倍数如 `2n/2n`）
4. 点击「重设图像」选择要处理的图片
5. 处理结果保存在 `data/picture2/` 目录

### 3. 记忆搜索

1. 在主界面点击「记忆搜索」进入搜索模块
2. 点击「刷新」加载 `data/memory/` 目录下的所有文本文件
3. 输入关键词搜索（空格分隔多个关键词），或输入文件编号直接读取内容到剪贴板
4. 点击「记忆文件夹」可打开文件存储目录

### 4. 批量处理

#### 批量重命名
1. 在批量处理界面设置文件头、位数、起始编号
2. 点击「选择文件」选择目标文件夹
3. 程序会按规则重命名文件夹内所有文件

#### 批量水印
1. 选择水印方式（文字水印 / 图片水印）
2. 设置水印内容、字体大小、透明度、位置（中间/左上/左下/右上/右下）
3. 点击「批量选择图片」选择包含图片的文件夹
4. 处理结果保存在 `data/picture2/` 目录

### 5. OCR 文字识别

1. 选择识别语言（中文/英文/日文/德文/韩文/法文）
2. 点击「选择图片」选择要识别的图片
3. 识别结果将显示在文本区域

> 注：OCR 模块当前未集成到主菜单，可直接运行 `python main6.py` 使用。

## 构建打包

使用 PyInstaller 打包为单目录可执行文件：

```bash
pyinstaller -D -w main.py
```

打包产物位于 `dist/自动化工具箱/` 目录。

## 技术栈

- **GUI 框架**：PySide6 (Qt for Python)
- **鼠标键盘控制**：pyautogui
- **图像处理**：OpenCV、Pillow
- **Excel 读取**：xlrd
- **OCR 识别**：PaddleOCR
- **剪贴板操作**：pyperclip

## 常见问题

1. **图片路径中包含中文导致读取失败**：请将图片和路径改为全英文路径
2. **配置文件格式错误**：请确保 `data/配置文件.xls` 格式正确，首行为表头，第二行为图片文件夹名
3. **OCR 模块依赖较大**：PaddleOCR 需要下载模型文件，首次使用可能需要较长时间
